# MIT License Change Boundary

ViennaPS `v4.3.0` and later are treated as post-MIT/GPL-era for this project.
The code in `jaxps` does not inspect or copy those source trees. Public behavior
from release notes and documentation can be re-specified in clean-room language
and implemented independently.

The implementation created in this repository should be reviewed against
`docs/after_mit_license_changes.md` and `docs/post_mit_feature_spec.md` before
future updates.
